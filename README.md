# 📊 Marketing Campaign Analysis

## Project Overview
Exploratory Data Analysis on a marketing campaign dataset of 2,240 customers
to understand spending behaviour, customer demographics and campaign performance.

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Charts Created
- Customer Education Level — Bar Chart
- Marital Status Split — Pie Chart
- Product Spending Analysis — Bar Chart
- Income Distribution — Histogram
- Campaign Acceptance Rate — Bar Chart
- Purchase Channel Comparison — Bar Chart
- Birth Year Distribution — Seaborn Histogram
- Income vs Wine Spending — Seaborn Scatter Plot
- Income Statistics Summary — NumPy

## Dataset
Marketing campaign data with 2,240 customers and 29 features including
income, spending habits, education, marital status and campaign responses.
